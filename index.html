Wij zijn DuckMC!

DuckMC is een minecraft server met het grootste aanbod van minigames aanbiedt! Daarnaast is onze server 24/7 online en hebben we al meer dan 1.000 unieke spelers! Dus waar wacht je nog op? Voeg het ip: play.duckmc.eu toe aan je server list en join!


Ontmoet onze community!

Onze server heeft een groot aantal spelers die zich elke dag weer voor uren kunnen vermaken met ons grote aanbod van games. Onze server heeft veel multiplayer gamemodes die je met onbekende of samen met je vrienden kunt gaan spelen, verbreek jij jouw vrienden hun records?

Speel ook op DuckMC!

Speel nu samen of alleen Bedwars, Party Games, The Bridge, Village Defense en/of 1 van onze vele andere toffe minigames die op onze server te bespelen zijn! Dan wensen wij jou alvast veel plezier!

1. Ga in Minecraft naar Multplayer
2. Klik op Add Server
3. Voer play.duckmc.eu in (zie voorbeeld) en klik daarna op Done
4. Selecteer onze server en speel gezellig mee!

Hoe werkt voten?

Ga naar de website van NameMC, voor een toffe beloning. Zorg dat je een account hebt op de NameMC website en koppel dit account aan je minecraft account.[Hoe claim ik mijn NameMC account?] Mocht je deze stappen voltooid hebben, vind je rechts boven een knop met een grote duim omhoog erop afgebeeld. Klik op deze knop, je eerst volgende keer dat je ons netwerk joint zal je de beloning ontvangen! Als alles goed gegaan in krijg je een bevestiging in de chat.



Java editie minecraft

Java editie is de originele versie van Minecraft! Java Edition kan op meerdere platforms tegelijk gespeeld worden op Windows, Linux en macOS. DuckMC ondersteunt enkel de java version van minecraft en is te bespelen in alle versies (1.8 t/m 1.16). De locatie van onze server is: Frankrijk (Europa).
Je kan meer informatie vinden over Minecraft Java Edition op de officiele Minecraft website: Breng me naar de officiele minecraft website.
